## こんにちは sharwood33sです。
Building practical AI tools for workflow improvement.
地方公務員として福祉業務に従事しながら、業務改善やAI活用に関心を持ち、IT分野の学習をしています。
将来的にはITへのキャリアチェンジを目指し、開発に取り組んでいます。
コードはまだまだ学習中ですが、AIコーディングツール（Codex）を活用しツールの作成をしています。

## 興味分野
- Pythonによる自動化
- ローカルLLMの活用
- 音声認識・議事録作成
- DX ／　業務改善
