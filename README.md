## 主な開発

### ローカルAI議事録作成ツール

福祉業務では他部署との連携を図るための会議が多く、そのたびに議事録作成に時間がかかっていました。
議事録作成を自動化し、本来の業務に集中できる時間を増やすためにこのツールを作成しました。

MLX-Whisper(mac) / faster-whisper(Windows)、pyannote.audio、LM StudioまたはOllamaを用いて、文字起こし、話者分離、要約を行うツールです。
マニュアル作成モードも搭載し、音声から手順書作成にもつなげられます。

- 初回のモデルロード時以外はオフラインで完結
- 文字起こし: MLX-Whisper / faster-whisper
- 話者分離: pyannote.audio
- 要約・議事録作成: LM Studio / Ollama
- GUIによる直感的操作

### 画像圧縮アプリ

ホームページ等に画像をアップロードする際、写真のサイズが大きすぎて使用できないことがありました。
せっかく撮った写真を無駄にしないために、Macのフレームワークを使った画像圧縮アプリを作成しました。

- 画像をドラッグ&ドロップで簡単に選択
- アスペクト比の指定、維持、拡大に対応
- 品質を調整してファイルサイズを圧縮

### サブスク管理アプリ

サブスクリプションの便利なサービスが増え、何にいくら使っているのかわからなくなってしまったため、このアプリを作成しました。
Next.jsで開発した、サブスクリプション支出の可視化ツールです。

- 毎月いくら払っているかを可視化
- 利用中、トライアル、解約予定、今週の請求件数を確認
- タグ付けや自動更新のON / OFFに対応
- 今後CSV出力機能を実装予定
