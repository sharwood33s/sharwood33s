## 主な開発
### ローカルAI議事録作成ツール
福祉業務では他部署との連携を図るための会議が多くあり、その度に議事録の作成に追われていました。
議事録作成を自動化し本来の業務に集中できるようにこのツールを作成しました。
MLX-Whisper、pyannote.audio、LM Studioを用いて文字起こし、話者分離、要約を行うツールを開発
マニュアル作成モードも搭載し、音声でマニュアル作成も可能
- 初回のモデルロード時以外はオフラインで完結
- 文字起こし：MLX-Whisper
- 話者分離：pyannote.audio
- 要約・議事録作成：LM Studio
- GUIによる直感的操作

### 画像圧縮アプリ
ホームページ等に画像をアップロードする際、写真のサイズが大きすぎて使用できないことがありました。
せっかく撮った写真を無駄にしないためにMacのフレームワークを使った画像圧縮アプリを作成しました。
- 画像をドラック&ドロップで簡単に選択
- アスペクト比の指定や維持、拡大も可能
- 品質を調整しファイルサイズの圧縮が可能に

### サブスク管理アプリ
サブスクの便利なサービスが増え、色々なサービスに課金しているうちに何にいくら使ったのかわからなくなってしまったので、
このアプリを作成しました。
Next.jsで開発したサブスクリプションの支出可視化ツールです。
- 毎月いくら払っているか可視化できる
- 利用中、トライアル／解約予定、無料トライアル、今週の請求の件数がわかる
- タグ付けや自動更新のON／OFFに対応
- 今後CSVで出力できる機能を実装予定
